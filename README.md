# Overview

This demo uses HTML, TypeScript and Webpack to showcase a simple sample for Amazon Chime SDK for JavaScript.

## Steps to run the demo

1. Clone this repo
```
git clone https://github.com/devalevenkatesh/vanilla-js-sdk-demo.git
```

2. Install NPM dependencies
```
npm install
```

3. Export your AWS credentials to the current session in local terminal.
```
export AWS_ACCESS_KEY_ID=<access-key-id>
export AWS_SECRET_ACCESS_KEY=<secret-access-key>
```

4. Start the local Node.js server
```
node server.js
```

5. In another terminal tab, start the Web app
```
npm start
```
